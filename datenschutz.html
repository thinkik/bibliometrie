Datenschutzhinweis

1. Allgemeine Hinweise zur Datenverarbeitung

Diese Website ist als statische Lernplattform umgesetzt. Es werden keine personenbezogenen Daten auf einem Server gespeichert, zusammengeführt oder für Profile verarbeitet.

2. Nutzung von LocalStorage

Zur Verbesserung der Bedienbarkeit verwendet die Website ausschließlich den LocalStorage deines Browsers. Dort werden lokal auf deinem Endgerät folgende Einstellungen gespeichert:

  • die gewählte Sprache (z. B. Deutsch/Englisch),
  • die ausgewählte Rolle,
  • dein Lernfortschritt innerhalb der Lerninhalte.

Diese Informationen bleiben auf deinem Gerät und werden nicht an den Betreiber der Website übertragen.

3. Cookies, Analyse und Werbung

Die Website setzt keine Cookies. Es werden keine Tracking- oder Analysewerkzeuge eingesetzt und keine Daten an externe Analyse- oder Werbedienste übermittelt.

4. Einbindung von D3.js über jsdelivr

Für einzelne Visualisierungen wird die JavaScript-Bibliothek D3.js über das Content Delivery Network (CDN) jsdelivr geladen. Beim Abruf dieser Bibliothek wird aus technischen Gründen deine IP-Adresse an den CDN-Anbieter übertragen, damit die Datei ausgeliefert werden kann.

D3.js steht unter der BSD-3-Clause-Lizenz: https://github.com/d3/d3/blob/main/LICENSE

5. Hinweise zu Download-Dateien (CSV/JSON)

Die bereitgestellten Download-Dateien (insbesondere CSV- und JSON-Dateien) sind für die lokale, offline nutzbare Weiterverarbeitung vorgesehen. Beim Öffnen oder Verarbeiten dieser Dateien werden keine Inhalte an einen Server dieser Website gesendet.

6. Kontakt bei Datenschutzfragen

Bei Fragen zum Datenschutz, zur Funktionsweise der Website oder zur lokalen Datenspeicherung kannst du dich über die im Impressum bzw. in den Projektunterlagen genannten Kontaktwege (z. B. E-Mail oder institutionelle Ansprechstelle) an das verantwortliche Team wenden.
